# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026

### Added
- 40+ mathematical operations
- Calculation history with timestamps (last 50 entries)
- Enhanced UI with Unicode box-drawing characters
- Emoji icons for visual appeal
- Statistics module (mean, median, max, min, range)
- Number system conversions (binary, octal, hexadecimal)
- Inverse trigonometric functions (arcsin, arccos, arctan)
- Advanced operations (factorial, GCD, LCM, nth root)
- Cube root function
- Natural logarithm (ln)
- Exponential function (e^x)
- Absolute value function
- Modulo operation
- Percentage calculator
- Degrees/Radians conversion
- Comprehensive test suite
- Demo script
- Multiple documentation files

### Enhanced
- Error handling for all operations
- Input validation with user-friendly messages
- Result formatting with proper precision
- Menu navigation system
- User prompts with clear indicators

### Fixed
- Division by zero handling
- Square root of negative numbers
- Logarithm domain validation
- Factorial of negative numbers
- Inverse trig function range validation

## [1.0.0] - Initial Release

### Added
- Basic arithmetic operations (add, subtract, multiply, divide)
- Scientific functions (power, square root, logarithm)
- Trigonometric functions (sin, cos, tan)
- Menu-driven CLI interface
- Basic error handling
